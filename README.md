# Ecommerce Website

A simple ecommerce website built using PHP, MySQL, HTML, CSS, and XAMPP.

## Features
- User Registration
- Login System
- Product Listing
- Add to Cart
- Logout Functionality

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- XAMPP

## Project Structure
- css/
- images/
- includes/
- pages/
- index.php

## How to Run
1. Install XAMPP
2. Move project to htdocs folder
3. Start Apache and MySQL
4. Open localhost/ecommerce
